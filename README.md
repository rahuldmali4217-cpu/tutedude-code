# To-Do List Application

## Features

- Create Task
- Update Task
- Delete Task
- Search Task
- Update Status
- MongoDB Storage

## Tech Stack

- React
- Node.js
- Express.js
- MongoDB

## Setup

### Backend

npm install

npm run dev

### Frontend

npm install

npm run dev

## Environment Variables

MONGO_URI=
PORT=

## Deployment

Backend: Render

Frontend: Netlify
